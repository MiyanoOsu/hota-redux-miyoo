# HEART-OF-THE-ALIEN  

(Another World 2) (Miyoo homebrew)

Port for Powkiddy Q90/V90

build from source: make

Keymap:
- A: use whip
- B: Jump
- Y: Run/shot
- X: hack speed
- Select: exit
- Start: Password screen
- L: quick save
- R: quick load
